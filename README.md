## Getting Started with `pst-client`

### Prerequisites for running `pst-client`

- `npm` 
- `git` 

### Running it locally

Clone this repository with `git`.

```powershell
git clone https://github.com/Project-SpareTime/client
```

 Once the repository is clone, change you current-working directory to client.

```powershell
cd client
```

 Now, you would want to install `npm` modules.

```powershell
npm install
```

 If you have yarn, type `yarn start` to run the server, otherwise use `npm start` to run the server.

### Pushing changes to the repository

The `Project-SpareTime/client` repository is hooked to a Netlify service with auto deploys enabled.

So on every successful commit merge, the changes get deployed via the Netlify CI/CD to the [site](https://pst-client.netlify.app/).
